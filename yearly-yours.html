Yearly Yours

Overview

Yearly Yours is a mobile app prototype designed to help users never miss an important birthday. Built in Figma as a collaborative team project, the prototype visualizes the full user workflow across four core features: profile creation, card sending, gift selection, and event planning.

My Role

This project was developed in a team of four, with each member owning the design of one primary user task. Working collaboratively in Figma, we designed our individual workflows to integrate seamlessly into a cohesive end-to-end experience.

Key Features

Profile Creation — Users onboard by creating a username, password, and personal profile. They can connect with friends via phone contacts or manual entry, and select personal interests which feed into the gift recommendation experience.

Card Sending — From the birthday reminder landing screen, users can send a personalized card directly to a friend. The card editor supports custom messages, stickers, hand-drawn illustrations, music, and photo attachments, delivered straight to the recipient's phone number.

Gift Selection — Users can browse gift recommendations tailored to their friend's stated interests. Items can be filtered by price and retailer, and selecting a gift redirects the user to the retailer's site to complete the purchase.

Event Planning — Users can create and manage birthday events by adding details such as event name, date, venue, and notes. Invitations are sent directly to selected friends, who can RSVP and view event details within the app.

Reflection

This project was my introduction to Figma and collaborative UX design. Coordinating workflows across a team taught me the importance of design consistency and clear component structure — ensuring that four independently designed flows felt like one unified product.
